# 🛍️ Kabayan Shop Saudi – E-commerce Website

## 📌 Description  
Kabayan Shop Saudi is a modern e-commerce website designed for selling stylish and affordable women’s clothing across Saudi Arabia.  
The platform provides a smooth shopping experience with easy product browsing, cart management, and quick checkout using WhatsApp integration.

This project is built to help small fashion businesses manage products and orders efficiently with a simple and scalable system.

---

## 🚀 Features  

- 🛒 Add to Cart & Checkout System  
- 📍 Location-based Delivery Charge (Riyadh, Jeddah, Dammam, etc.)  
- 📱 WhatsApp Order Integration (Direct Order System)  
- 👗 Dynamic Product Listing from Strapi CMS  
- 🔐 Clean UI & Fully Responsive Design  
- 📦 Order Data Storage (Database / Google Sheets option)  
- ⚡ Fast Loading & Optimized Performance  

---

## 🛠️ Technologies Used  

- ⚛️ React.js  
- 🎨 Tailwind CSS  
- 🌼 DaisyUI  
- 🔌 Strapi (Headless CMS Backend)  
- 🌐 REST API  
- ☁️ Netlify (Frontend Hosting)

## 📦 How It Works  

1. Browse available products  
2. Select size, color, and quantity  
3. Add items to cart  
4. Enter delivery details (location, phone, etc.)  
5. Delivery charge is calculated based on selected city  
6. Place order via WhatsApp or backend system
